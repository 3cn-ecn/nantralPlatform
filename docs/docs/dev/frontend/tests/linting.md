# Linting and Formatting

Enforces a consistent coding style across the project, and catches bugs before
they come.

## On the Front End

- Linter: we use [eslint](https://eslint.org/)
  ```bash
  npm run lint
  ```
- Formatter: also [eslint](https://eslint.org/)
  ```bash
  npm run lint:fix
  ```
